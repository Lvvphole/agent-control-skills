# Handoff

This package is a specification for an Agent Skills library. Install by copying `agent_control_skills/` as an isolated skill package after review. Do not run implementation, deployment, shell scripts, or root-file edits from this handoff.

Review order: manifest, frontmatter audit, trigger matrix, boundary matrix, dependency graph, security checklist, test checklist, then each `SKILL.md`.

Use skills only when their descriptions match the user request. No skill grants completion authority; release claims require verifier PASS, evaluator PASS, complete evidence, and zero unresolved escalations.
